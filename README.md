# Fullstack IM_Service

##To start:
### For the frontend:
  1. `cd im-service`
  2. `npm i`
  3. `npm run dev`
### For the backend:
  1. `docker-compose up --build`
### To connect to the database:
  1. Download MySQL Workbench
  2. Make a new connection
